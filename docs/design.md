1. Color & Background
- Full-screen soft gradient backgrounds
- Gradients are low contrast
- Dominant colors:
- Warm beige
- Soft peach / blush
- Light gray
- No pure white backgrounds
- No pure black backgrounds (except loading screens)
- Gradient direction is subtle (radial or very slow vertical)

2. Contrast & Hierarchy
- Text contrast is intentionally low
- Primary CTA uses solid black to stand out
- Secondary text uses light gray
- No bright accent colors
- No visual separators (no lines, dividers)

3. Shape Language
- Everything is rounded
- Buttons: fully pill-shaped
- Cards / bubbles: large corner radius
- No sharp corners anywhere
- Icons are soft, abstract, non-representational

4. Layout Patterns
- Single-column layouts only
- Center-aligned content on intro screens
- Bottom-pinned primary CTA
- Generous vertical spacing
- Very few elements per screen (2–4 max)

5. Typography (observable)
- Large headline text
- Wide line spacing
- Short line length
- Serif-like or humanist font for headings
- Simple sans-serif for body
- No bold-heavy text
- No all-caps

6. Iconography
- Icons are soft, abstract, non-representational
- Icons are small and subtle
- Icons are used sparingly
- Icons are consistent across the app

7. Motion & Loading Indicators
- Loading states are full-screen
- Animated dot cluster instead of spinners
- Animation is soft blur / fade, not rotation
- No progress bars
- No percentage indicators

8. Input Fields
- Text inputs are:
- Large
- Rounded
- High padding
- Placeholder text is light gray
- Focus state is subtle (no strong outline)
- Keyboard covers lower part of screen naturally (no avoidance UI)

9. Chat / Content Cards
- Messages appear as floating cards
- Cards have:
- Rounded corners
- Light gray background
- Cards are separated by large vertical spacing
- No speech-tail bubbles
- Date headers are muted and minimal

10. Paywall / System UI Treatment
- Paywall visually matches main app
- No “hard” modal borders
- Apple system sheets shown as-is
- Subscription CTA remains visually minimal
